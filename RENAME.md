# 💰 Finance Dashboard UI

## 📌 Project Overview

This project is a modern **Finance Dashboard UI** built using **HTML, CSS, and JavaScript**.
It provides users with a complete overview of their financial activities including income, expenses, analytics, and savings goals.

The dashboard focuses on clean design, interactive components, and dynamic data handling without any backend dependency.

---

## 🎯 Objective

To design and develop a responsive and interactive finance dashboard that demonstrates:

* UI/UX design skills
* Frontend development concepts
* Data handling using JavaScript

---

## 🚀 Features

### 🏠 Dashboard

* Displays **Total Balance, Income, Expenses, and Savings Rate**
* Interactive **Bar Chart and Pie Chart**
* Monthly overview (March)

### 💸 Transactions

* Add income and expense transactions
* Delete transactions
* Dynamic update of dashboard data
* Data stored using **Local Storage**

### 📊 Analytics

* Line chart showing **Income vs Expense trends**
* Category-based **expense distribution**

### 📈 Reports

* Visual representation of financial data
* Expense breakdown and comparisons

### 🎯 Savings Goals

* Tracks savings progress toward a goal (₹50,000)
* Dynamic progress bar

### 👤 User Profile

* Clickable user name
* Displays:

  * Account details
  * Bank information
  * Debit & Credit card UI

### 📤 Export Data

* Export transaction data as a **CSV file**
* Compatible with Excel / Google Sheets

### 🌙 Dark Mode

* Toggle between light and dark themes

---

## 🛠️ Technologies Used

* **HTML5** – Structure
* **CSS3** – Styling & Layout
* **JavaScript (ES6)** – Logic & Interactivity
